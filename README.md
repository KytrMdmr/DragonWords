# DragonWords

DragonWordsは、右Ctrlキーを押している間だけ音声を録音し、キーを離すと日本語の音声認識を行い、認識結果をクリップボード経由で自動入力するWindows向け音声入力補助アプリです。

## 概要

文章入力を効率化するために作成した、個人開発の音声入力ツールです。

小さなステータスウィンドウを画面右上に表示し、現在の状態を色と文字で分かりやすく表示します。

## 主な機能

- 右Ctrlキーを押している間だけ録音
- キーを離すと音声認識を開始
- Google音声認識による日本語文字起こし
- 認識結果をクリップボードへコピー
- 現在選択中の入力欄へ自動ペースト
- 画面右上に常駐する小さなステータスウィンドウ
- PyInstallerによるEXE化に対応

## 使い方

1. アプリを起動します
2. 入力したいテキスト欄をクリックします
3. 右Ctrlキーを押しながら話します
4. 右Ctrlキーを離すと音声認識が始まります
5. 認識された文字が自動で貼り付けられます

## 注意点

- 音声認識にはインターネット接続が必要です
- Googleの音声認識機能を使用しています
- 常時録音は行わず、右Ctrlキーを押している間のみ録音します
- キーボード入力の監視、自動ペースト、クリップボード操作を行います
- Windowsでの利用を想定しています
